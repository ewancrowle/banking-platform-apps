# Banking Platform Apps 💸

This repository houses the applications for my Individual Project (CM3203) at Cardiff University. The project involves the research, design, and development of a fully functional, open-source online banking system, comprising a microservices backend, a Kubernetes cluster, and a mobile app, with research focusing on backend development.

Configurations, Kubernetes manifests, and instructions for deploying the apps to Kubernetes are maintained separately in the [banking-platform-gitops](https://github.com/ewancrowle/banking-platform-gitops) repository.
